== Description ==

Add a required policy checkbox to any comment forms and a custom link to the policy polity text opening in new tab.
Features:
  - Internationalized
  - Compatible with Polylang
  - Translated to `es_ES` and `ca`

== Installation ==

1. Upload `wpcpc` folder to the `/wp-content/plugins/` directory or install from 'Plugins' menu in WordPress.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Set the polity policy page in 'Settings/Comments' menu in WordPress.

== Screenshots ==

1. Setting field on comments setting page.
2. Policy checkbox and link printed on the bottom of the comment form, before the submit button.
2. Error message if the policy checkbox is not checked.
